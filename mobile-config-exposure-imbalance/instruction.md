Write /app/solve.py that reads a binary config-dependency file (path is the first argument) and prints one number: the total value of the largest group of configs that all mutually depend on each other, counting only groups reachable from the root config, id 0.
The little-endian .mcfg is: magic "MCFG", a uint32 version, a uint32 count, a uint32 data_offset, then count records of {int32 id, int32 value, uint32 dep_count, then dep_count uint32 dependency ids}; a sample is at /app/data/scene.mcfg.
Parse it yourself: no numpy, scipy, networkx, igraph or other graph/array library, and no shelling out. We grade on files you have not seen.
